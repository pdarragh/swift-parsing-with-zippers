ExpressionCase

public indirect enum ExpressionCase : Equatable

ExpressionCases encode the particular type of Expression being dealt with.

  • Tok(token:)

    Tokens are trivial constructions.

    Declaration

    Swift

    case Tok(token: Token)
  • Seq(symbol:expressions:)

    Sequences consist of a Symbol (naming the production represented by the sequence) and a sequence of Expressions.

    Declaration

    Swift

    case Seq(symbol: Symbol, expressions: [Expression])
  • Alt(expressions:)

    Alternates are not named. They contain a collection of child Expressions.

    NOTE: The parsing implementation requires that this array can be updated in-place, so we use a ReferenceArray instead of a simple array.

    Declaration

    Swift

    case Alt(expressions: ReferenceArray<Expression>)
  • ==(_:_:)

    ExpressionCases are trivially Equatable from their components.

    Declaration

    Swift

    public static func == (lhs: ExpressionCase, rhs: ExpressionCase) -> Bool
